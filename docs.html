Setup

Installation Roadmap

Follow these steps to deploy Yuros Blackbird Project on your machine.

1. Flashing the ISO

Download the latest release and use a tool like Rufus or balenaEtcher.

  • Select the downloaded .iso file.
  • Choose your USB Drive (Min 8GB).
  • Click Flash and wait for completion.

2. Entering BIOS/UEFI

Reboot your PC. Continuously press F2, F12, or Del to enter BIOS settings. Change the Boot Priority to boot from the USB Drive first.

3. Text-Based Setup

Yuros Blackbird uses a highly reliable text-based installer. Use your arrow keys to navigate the menu and follow the prompts to format your drive and install the base system.

Academic Workflow

Student Guide

How to utilize the automated terminal recording for your assignments.

Fetching CLI Logs

Every command you type is captured by the background daemon. To fetch your session log:

cat /var/log/yuros_student_cli.log > ~/assignment_logs.txt

Packaging for Submission

Zip your logs alongside your metadata analysis output to submit to the lecturer portal.

tar -czvf assignment_submission.tar.gz ~/assignment_logs.txt ~/mods_data.xml

System Logs Compilation

When reporting bugs, always include the core trace logs. Retrieve them using our built-in debug script:

sudo yuros-debug --export-tar
